# BarberShop API

> Backend API for BarberShop application, which is an appointment manager for a barber shop

## Usage

Rename ".env.env" to ".env" and update the values/settings to your own

# Install Dependencies

```
npm install
```

## Run App

```
# Run in dev mode
npm run dev

# Run in prod mode
npm start
```
